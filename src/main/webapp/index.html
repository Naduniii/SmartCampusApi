Smart Campus API

Base URL: /api/v1

Note: Use Postman or curl for POST and DELETE requests.

Note: Replace path variables such as {roomId} or {sensorId} with actual IDs.

Endpoints

GET /api/v1 - API discovery endpoint (system metadata)
GET /api/v1/rooms - Retrieve list of all rooms
POST /api/v1/rooms - Create a new room in the system
GET /api/v1/rooms/{roomId} - Get details of a specific room by ID
DELETE /api/v1/rooms/{roomId} - Delete an existing room
GET /api/v1/sensors - Retrieve all registered sensors
POST /api/v1/sensors - Register a new sensor
GET /api/v1/sensors?type=CO2 - Filter sensors by sensor type
GET /api/v1/sensors/{sensorId}/readings - Retrieve readings for a specific sensor
POST /api/v1/sensors/{sensorId}/readings - Add a new reading for a sensor
